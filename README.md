QRZ.com XML Data Service client for amateur radio callsign related lookups in Elixir

## Installation

The package can be installed by adding qrz to your list of dependencies in mix.exs:

```elixir
def deps do
  [
    {:qrz, "~> 0.1.0"}
  ]
end
```
